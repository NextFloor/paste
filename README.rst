Paste
=====

*Paste* is a simple pastebin built on top of *flask*.


Dependency
----------

* python >= 3.4.0
* see ``requirements.txt``


How to run
----------

.. code-block:: console

   $ git clone https://github.com/nextfloor/paste.git
   $ cd paste
   $ pip install -r requirements.txt
   $ FLASK_APP=paste flask run
